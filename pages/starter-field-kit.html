Separate from merch

Admin plus physical presence.

Deployment gear is about support bundles and visible culture objects. The starter field kit is about how a local crew shows up, listens, records, follows up and keeps public/private boundaries clear.

Pattern, not mandate

Local crews need a repeatable starter pattern.

A branch, electorate group or community crew can begin with a table, a banner, QR cards, name badges, stickers, a local help table, coffee chats, AI demos, phone mounts, a media kit, shared folders, social posts and a small landing page.

Strange But True is Luke's personal sole trader business and can be used as one pattern library for practical field setup. It is not mandatory infrastructure for P4A organisers, branches or communities.

01

Admin spine

Shared folders, source notes, contact lists, run sheets, correction logs, volunteer role cards, consent notes and a simple handover pack so the crew is not trapped in one person's head.

02

Market stall presence

Folding table, banner, printed QR cards, name badges, stickers, sign-up sheet, local issue cards, water, shade, basic safety kit and a clear local help offer.

03

Media basics

Phone mounts, tripod, small mic kit, event photo checklist, short video templates, social media captions, local landing page links and a plain review process before posting.

04

Local help table

Start with useful help: grant forms, tech support, public noticeboards, civic questions, local asset notes, issue intake and referrals. Trust grows through repeated visible wins.

05

Public and private streams

Public profile.md and ledger material stays opt-in and reviewable. Private notes, private help, home details, personal files and sensitive contact trails stay private by default.

06

Compliance guardrails

Label the page as exploratory where needed, avoid fake endorsements, check local event rules, respect electoral law, and keep public claims source-linked and correctable.

Useful reference patterns

These are nearby pattern libraries, not compulsory templates. They show how local pages, noticeboards and profile kits can make a community layer visible without turning the whole thing into a top-down franchise.
